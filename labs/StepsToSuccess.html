Success

Success is achieving satisfaction in completing a set goal!. Here is a recipe for achieving success.

Ingredients

  • Project
  • Requirements
  • Plan for achieving success!

Method

  1. Clearly access what needs to be done for the project.
  2. Device a step by step plan on achieving success in the endeavor.
  3. Acquire all requirements - Technical skills and materials.
  4. Carefully implement each step of the plan in completing the project
  5. Go over project, note all corrections and updates needed, make these corrections and updates, recheck!
  6. When all is satisfying, Declare project a Success!
